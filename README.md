# Marine Microbiome Project

## Dataset Information

Dataset Name: SRR37876396

Source: NCBI SRA

Type: 16S rRNA Amplicon Microbiome

Sample: Seawater Microbiome (W10)

Download Size: 134.5 MB

## Objective

To analyze marine microbiome sequence data and construct a phylogenetic tree.

## Workflow

1. Download microbiome dataset from NCBI SRA
2. Store and document data in GitHub
3. Quality Control (QC)
4. Convert FASTQ to FASTA (if needed)
5. Multiple Sequence Alignment using Clustal Omega
6. Construct phylogenetic tree using MEGA
7. Interpret phylogenetic relationships

## Software

- GitHub
- Clustal Omega
- MEGA
- NCBI SRA
